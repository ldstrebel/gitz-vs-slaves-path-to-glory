# Core Rules: Commands (AoS 4th Edition)

## 1.0 Commands Overview

**Command Points (CP)** are a resource used to activate specific abilities.

- **Cost:** Any ability with a number in the top right (e.g., 1 or 2 is a Command. You must pay the CP cost to use it.  
- **Usage Limits:**  Each **Unit** can only receive 1 Command per phase.  
  - Each **Command Ability** can only be used 1 time by the army per phase.

### 1.1 Earning Command Points

- **Start of Battle Round:** Each player gains **4 CP**.  
- **Underdog Bonus:** The Underdog gains **1 CP** (Total 5).  
- **Reset:** At the end of the Battle Round, all unspent CP are lost (reset to 0).

---

## 2.0 Hero Phase Commands

### **Rally** (1 CP)

- **Timing:** Any Hero Phase  
- **Declare:** Pick a friendly unit **not in combat**.  
- **Effect:** Roll 6 dice. For each **4+**, gain **1 Rally Point**.  
  - Spend 1 Point: **Heal (1)**.  
  - Spend Points equal to Health Characteristic: **Return** a slain model.  
  - *Unspent points are lost.*

### **Magical Intervention** (1 CP)

- **Timing:** Enemy Hero Phase  
- **Declare:** Pick a friendly **WIZARD** or **PRIEST**.  
- **Effect:** The unit may use a **Spell** or **Prayer** ability immediately.  
  - **Penalty:** Subtract 1 from the casting or chanting roll.

---

## 3.0 Movement Phase Commands

### **Redeploy** (1 CP)

- **Timing:** Enemy Movement Phase  
- **Declare:** Pick a friendly unit **not in combat**.  
- **Effect:** Each model in the unit moves up to **D6"**.  
  - Cannot pass through or end within Combat Range of an enemy.  
  - *Keywords:* **MOVE**, **RUN**

### **At The Double** (1 CP)

- **Timing:** Reaction (You declared a **RUN** ability)  
- **Used By:** The unit using the Run ability.  
- **Effect:** Do **not** make a run roll. Instead, add **6"** to the unit's Move characteristic for that move.

---

## 4.0 Shooting Phase Commands

### **Covering Fire** (1 CP)

- **Timing:** Enemy Shooting Phase  
- **Declare:** Pick a friendly unit that is **not in combat** and has **not Run** this turn. Pick the closest eligible enemy unit as the target.  
  - *Restriction:* Cannot target Manifestations or Faction Terrain.
- **Effect:** Resolve shooting attacks against the target immediately.  
  - **Penalty:** Subtract 1 from **Hit rolls**.  
  - *Keywords:* **SHOOT**, **ATTACK**

---

## 5.0 Charge Phase Commands

### **Counter-Charge** (2 CP)

- **Timing:** Enemy Charge Phase  
- **Declare:** Pick a friendly unit **not in combat**.  
- **Effect:** That unit can use a **CHARGE** ability as if it were your charge phase.

### **Forward to Victory** (1 CP)

- **Timing:** Reaction (You declared a **CHARGE** ability)  
- **Used By:** The unit using the Charge ability.  
- **Effect:** You can **re-roll** the charge roll.

---

## 6.0 Attacking Commands (Combat & Shooting)

### **All-Out Attack** (1 CP)

- **Timing:** Reaction (You declared an **ATTACK** ability)  
- **Used By:** The unit using the Attack ability.  
- **Effect:**  **Add 1** to **Hit rolls** for that ability (affects Companion weapons).  
  - **Drawback:** For the rest of the turn, **subtract 1** from **Save rolls** for the unit using this ability.

---

## 7.0 Defensive Commands

### **All-Out Defence** (1 CP)

- **Timing:** Reaction (Opponent declared an **ATTACK** ability)  
- **Used By:** A unit targeted by that Attack ability.  
- **Effect:** **Add 1** to **Save rolls** for that unit until the attack ability is resolved.

---

## 8.0 End of Turn Commands

### **Power Through** (1 CP)

- **Timing:** End of Any Turn  
- **Declare:** Pick a friendly unit that **Charged** this turn. Pick an enemy unit in combat with it (Target must have lower Health characteristic).  
- **Effect:** 1  Inflict **D3 Mortal Damage** on the target. 2  Friendly unit can move up to its **Move** characteristic. 3  It can move through Combat Ranges of enemies it was in combat with, but not others. It does not have to end in combat.  
  - *Keywords:* **MOVE**
