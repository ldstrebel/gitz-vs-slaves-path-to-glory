# Character Development — The Wake of the World-Eater

*Per-unit record of Path abilities and scar history. The Battle Bible holds current state; this file holds the full journey.*
*Updated by the Aftermath Gem after each battle.*

---

## How to Read This File

**Path Abilities** — every ability a unit has ever earned, in order, with the battle it was earned and the tier it belongs to.
**Scar History** — every scar a unit has ever received, when it was earned, and whether it was later cleared by healing.

Scars in this campaign are **not permanent** — they clear if wounds are healed below the threshold. A scar marked `[CLEARED]` was real, it affected play, and it's gone now.
